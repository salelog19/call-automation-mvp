# План: Настройка бесплатных ИИ-моделей в Kilo

## Цель
Настроить Kilo Code для работы с бесплатными ИИ-моделями через конфигурационный файл `kilo.json`.

## Шаги

1. **Создать файл `kilo.json` в корне проекта** (`C:\ВАЖНО\VS CODE\kilo.json`)

Конфигурация:
```json
{
  "model": "kilo-auto/free",
  "small_model": "kilo-auto/free",
  "disabled_providers": []
}
```

2. **Альтернативные бесплатные модели** (можно выбрать любую):
   - `kilo-auto/free` — автовыбор лучшей бесплатной модели (рекомендуется)
   - `bytedance-seed/dola-seed-2.0-pro:free`
   - `x-ai/grok-code-fast-1:optimized:free`
   - `openrouter/free`

3. **Для использования BYOK (свой ключ Mistral)**:
```json
{
  "model": "mistral/codestral",
  "small_model": "mistral/codestral",
  "provider": {
    "mistral": {
      "options": {
        "apiKey": "ваш-ключ-mistral"
      }
    }
  }
}
```

## Результат
После создания файла Kilo будет использовать бесплатные модели для генерации кода и фоновых задач.
