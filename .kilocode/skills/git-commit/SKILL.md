---
name: fast-deploy
description: Быстрый деплой без лишних вопросов
-----------------------------------------------

Когда пользователь пишет:

деплой

или

push

выполняй ТОЛЬКО:

1. cd backend && npm run build

2. Если build успешен:

* git add .
* git commit -m "update"
* git push

3. После git push:

* сообщи пользователю что нужно нажать Redeploy в Coolify

4. НЕ:

* сканируй другие папки
* выполняй tree
* проверяй весь ПК
* ищи файлы вне workspace
* задавай лишние вопросы
* делай дополнительные git команды

5. Работай только внутри текущего workspace проекта.
6. Сообщай результат КРАТКО:

* build ok
* pushed
* нажмите Redeploy в Coolify
