---
name: safe-deploy
description: Делает безопасный деплой (с проверкой билда)
---

Когда пользователь пишет "задеплой":

1. Выполни:
npm run build

2. Если есть ошибка:
- остановись
- покажи ошибку
- НЕ делай git push

3. Если build успешен:
- git add .
- git commit -m "update"
- git push

4. Сообщи "деплой отправлен"
